一款简洁、高效的梅赛德斯-奔驰汽车落地价计算工具，帮助您快速估算购车总成本。

## ✨ 功能特性

- **车型选择**: 内置多款热门奔驰车型及其官方指导价。
- **精准价格计算**:
    - **购置税**: 根据最新税率自动计算。
    - **保险费用**: 包含交强险和商业险，并支持自定义金额。
    - **上牌费**: 可自定义设置。
- **灵活自定义**:
    - 支持输入**厂商优惠**和**经销商优惠**。
    - 允许调整**商业保险**费用。
- **明细清晰**: 自动汇总所有费用，生成详细的落地价构成清单。
- **移动端优先**: 完美适配手机、平板等移动设备，随时随地轻松使用。
- **数据持久化**: 自动保存您上次输入的数据，方便下次访问。
